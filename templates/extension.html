🧩 Job Application Tracker Extension

Track your job applications directly from your browser — simple, fast, and private.

This lightweight extension helps you log and monitor job applications right from job boards, LinkedIn, or any job portal you use.

⚙️ How to Install

Install the extension manually using Developer Mode. It takes less than 2 minutes.

👉 View Step-by-Step Installation Guide

📌 Features

  • 📝 One-click job tracking from any site
  • 💾 Data saved locally — nothing is uploaded anywhere
  • 🧠 No account, no login — just use it
  • 🔄 Update it easily through GitHub whenever needed

📍 Pin the Extension

  1. Click the 🧩 Extensions icon in Chrome (top right)
  2. Find Job Application Tracker in the list
  3. Click the 📌 pin icon next to it
